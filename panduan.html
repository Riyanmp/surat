Tentang Sistem

Sistem Persuratan Digital Ruangan ITBAN IV Inspektorat Provinsi NTB digunakan untuk mencatat, memantau, dan mengelola surat masuk secara digital. Semua data tersimpan secara lokal di browser (localStorage).

Cara Penggunaan
1
Login

Masukkan username dan password. Admin: admin/admin123, User: user/user123.

2
Input Surat Baru

Klik menu "Input Surat", isi semua kolom yang diperlukan (sumber, nomor, perihal, tanggal). Upload file surat jika ada (PDF/Word/Gambar, maks 2MB).

3
Kelola Data Surat

Di menu "Data Surat", gunakan pencarian untuk menemukan surat. Klik tombol "Ubah" untuk mengganti status. Klik ikon mata untuk melihat file lampiran.

4
Cetak Data

Klik tombol "Cetak" di halaman Data Surat untuk mencetak seluruh data dalam format tabel rapi.

5
Lihat Laporan

Menu "Laporan" menampilkan grafik surat per bulan, sumber surat terbanyak, dan rekap status dengan persentase penyelesaian.

6
Arsip Surat

Surat yang statusnya "Ditangani" otomatis masuk ke arsip. Anda juga bisa mengarsipkan manual dari menu "Arsip".

Kontak

Untuk bantuan atau kendala teknis, hubungi administrator sistem Ruangan ITBAN IV Inspektorat Provinsi NTB.
